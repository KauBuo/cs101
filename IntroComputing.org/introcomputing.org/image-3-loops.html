图形编程: 循环


  • 上图 flowers.jpg 457 pixels 宽 by 360 pixels 高 -- 164,520 pixels
  • 一个像素一个像素的操作，太不实际了
  • 我们用高级for循环来实现

高级for循环

  • 语法: for (p: image) { ...对p的操作... }
  • 以上代码表示循环 图形image中的每个像素
  • 循环到一个像素，就存到p里面，然后就对p做一些操作
  • 然后再循环下一个像素，再存到p里面，再对p同样的操作
  • 以此类推，知道把image中的所有像素p全部操作完成
  • 注意image只是一个存储有图形的变量名称
