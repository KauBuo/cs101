Image 2 Exercises

For these exercises, write code to manipulate and print the image "x.png". Some basic starter code to load and print the image is provided. Modify and add lines to the starter code as needed to solve each exercise.

1. Write code to set the pixel at (0, 0) to blue (i.e. blue value is 255, red and green are left at 0).


2. Write code to set the pixel at (0, 0) to be violet (i.e. red and blue both 255, green left at 0).


3. Write code to set the pixel at (1, 0) to be red.
