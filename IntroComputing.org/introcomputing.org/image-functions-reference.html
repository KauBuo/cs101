图像处理函数参考

image = new SimpleImage("flowers.jpg"); 让变量 image 保存 "flowers.jpg" 图像
image.setZoom(5); 把image保存的图像放大5倍
print(image); 打印image保存的图像
pixel = image.getPixel(0, 0); 获取image保存图像的第(0,0)点的像素，并保存到变量pixel中
print(pixel); 打印像素rgb值 "r:200 g:12 b:166"
pixel.setRed(200); 把pixel所保存像素的红色阶的值改为200，如果输入数值超过0~255的范围，则自动转化成0或者255
red = pixel.getRed(); 获得pixel中保存像素的红色阶的值
image.getWidth(), image.getHeight() 获得image中保存图像的宽width和高height
image.setSize(width, height); 把image中保存图像的宽和高改成新的值
image.setSameSize(image2); 以image2位标准，扩展或者收缩image。保证image的长宽比不变，并且最少和image2一样大
pixel.getX(); 获得一个像素点的横坐标
pixel.getY(); 获得一个像素点的纵坐标
