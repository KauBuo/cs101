数字图像基础知识

数字图像

  • 以下数字图像看似很清晰
  • 实际上它是由无数小方块组成的

像素

  • 放大10倍，既可以看到这些小方块
  • 学称: 像素 "pixels"
  • 每个像素只有一种颜色
  • 这个图片宽800像素，高600像素
  • 所以一共有 800 * 600 = 480,000 像素
  • 或者称之为 0.5 百万像素 megapixels
  • 如果一个图片是 8000 * 6000 的，那么就基本上属于 5百万像素了
  • 你的手机是多少像素呢?

像素定位

  • 以下坐标系用来定位像素
  • 图片左上角是(0,0)
  • x轴向右发展,y轴向下发展
  • 比如第一个像素点就是：像素点 (1,0)

如何造颜色

  • 牛顿的三棱镜试验告诉我们，颜色可以可以被分开

  • 我们就可以反向把颜色合成

  • 每一个像素点都有三个颜色的灯
  • 红色R,绿色G,蓝色B
  • 通过调节RGB每个小灯的亮度，就可以合成出任何一种整体的颜色的像素点,进而构成整个画面

RGB探索工具

用这个工具看看到底不同亮度的RGB混合的效果RGB探索工具

RGB - 3个数

  • 从探索工具可以看出 RGB分别被表示成了三个数
  • 每一个数字从 0~255
  • 0最暗，255最亮
  • 这样，不同的数字混合就能实现不同的效果了
颜色 R G B
红色 255 0 0
紫色 255 0 255
暗黄色 100 100 0
白色 255 255 255
黑色 0 0 0
