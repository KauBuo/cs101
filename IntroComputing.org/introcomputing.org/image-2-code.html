图像处理的工具：显示 + 放大

  • 3行代码
  • img = new SimpleImage("x.png");
  • 定义一个变量img代表x.png
  • 注意要声明 x.png 是一个新的new，简单的图像SimpleImage
  • img.setZoom(20);
  • 通过 img 的 set设置Zoom焦距方法，把x放大20倍
  • print(img);
  • 打印出来被放大20倍以后的图像x

图像处理的工具：获得像素点 + 改变像素点的RGB值

  • p = img.getPixel(0, 0);
  • get获得，Pixel像素 => 以上代码获得x的第(0,0)位置的像素点，并存入p中
  • p.setRed(255);
  • set设置，Red红 => 以上代码把获取的像素点的R值改为255


设置RGB的方法

  • p.setRed(number);
  • p.setGreen(number);
  • p.setBlue(number);
